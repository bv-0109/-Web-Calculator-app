# -Web-Calculator-app

**TASK **
Project: Web Calculator App

## Overview
This project is a simple calculator built using the Gradio framework.
It replicates the functionality of a basic calculator with buttons for digits (0–9),
operators (+, -, *, /), decimal point, equals (=), and clear (C).
The calculator supports both integer and float operations.

## Why Gradio?
Due to storage limitations on my system, I was unable to install heavy code editors
like Visual Studio Code or Jupyter Notebook locally.
Tkinter, which is commonly used for GUI projects in Python, requires a desktop environment
that is not available on web-based platforms like Google Colab.

To overcome this, I used **Gradio**, a lightweight web framework that allows building
interactive GUIs directly in the browser. This provides a Tkinter-like environment
without the need for local installations.

## Features
- Perform addition, subtraction, multiplication, and division.
- Supports float numbers (e.g., 3.5 + 2.1).
- Clear button to reset the display.
- Runs seamlessly in Google Colab or any web-based Jupyter environment.

## How to Run
1. Open Google Colab.
2. Copy-paste the provided Python code into a cell.
3. Run the cell.
4. A link will appear — click it to open the calculator in your browser.

## Dependencies
- Python 3.x
- Gradio (`pip install gradio`)

## Conclusion
This project demonstrates how Gradio can be used as a practical alternative to Tkinter
for building GUIs in web-based environments. It is lightweight, easy to implement,
and ideal for situations where installing heavy code editors is not possible.
